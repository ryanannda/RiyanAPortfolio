Cleaning Data

Riyan Ananda P31 Jan 2025

Microsoft Excel - Cleaning Data

Data cleaning is the process of cleaning, repairing, and restructuring a dataset to make it more accurate, consistent, and ready for analysis or modeling. This is one of the most important steps in data preprocessing because dirty data can result in incorrect analysis or inaccurate models.

Data Cleaning Steps

  • Handling Missing Values

    Ensure there are no missing data points. Missing values are often represented as NaN, NULL, or undefined entries. Depending on the context, missing values can be removed, replaced with mean, median, mode, or imputed using other techniques

  • Handling Duplicate Data

    Check for rows that contain identical data, which might be redundant and unnecessary. Remove rows with duplicate values where applicable.

  • Identifying and Handling Outliers

    Outliers can distort the analysis, so it is important to detect them. Outliers can be removed, replaced with more appropriate values, or treated using robust techniques.

  • Ensuring Data Consistency

    Ensure data formats are consistent, such as date, time, or numerical formats. Ensure categorical data has consistent labeling (e.g., avoiding variations such as "Male" vs. "male").

  • Correcting Incorrect Data Types

    Ensure each column has the correct data type, such as numerical columns for numerical data and categorical columns for categorical data.

  • Data Transformation (If Necessary)

    Occasionally, may need to perform normalization, standardization, or other transformations to make the data more suitable for analysis or modeling.

  • Validating the Data

    After the cleaning process, ensure that the resulting data is consistent and valid for further analysis.
