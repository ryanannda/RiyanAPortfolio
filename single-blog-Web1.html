Web - System Management Inventory

Inventory Management System is an application designed to help businesses manage stock efficiently. This application records, monitors, and manages incoming and outgoing goods, optimizes inventory, and prevents excess or shortage of stock. With features like these, this application can improve operational efficiency, reduce human error, and support data-based decision making.

Notes

Database and relations used can be seen on: Database

This website is built with the main programming language PHP and is connected to the SQL database, Some of the functions contained in the inventory management system website include:
- Provides features to add, edit, and delete item data in real-time.
- Group items based on certain categories to make searching easier.
- Displays the number of stock items accurately with automatic updates when there are incoming or outgoing transactions.
- Provides a complete list of transactions that can be filtered by date, category, or user.
- Assists in creating invoices or notes for each transaction that occurs.
- Role-Based Access Control (RBAC): Determines access rights based on user roles (admin, warehouse staff, cashiers, etc.).
- Can be used on desktop and mobile devices with a responsive display.
